## About Random Password Generator

built a random password generator application using HTML,CSS and javascript. it let the user to generate a random password and he can copy it to the clipboard with a single click

## How It Works

1. The user clicks the **Generate** button.
2. A random password is created using JavaScript.
3. The generated password appears in the input field.
4. The user can click the copy icon to copy the password.
5. A small notification appears confirming that the password was copied.

## Features

- Generates a Unique Random Password
- password can copied with a click
- a nice popup notification that alert the users about password copied confirmation
- message notification automatically disappers after few seconds
